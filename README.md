# Reproducible Brain Charts Demonstration & Workshop

This repository contains example code to work with RBC data. This was prepared for Reproducible Brain Charts (RBC) Demonstration & Workshop (March 19, 2026) for MindCORE at the University of Pennsylvania. The presentation is also included in this repository.

More informatio on the RBC dataset and how to download it is available at [https://reprobrainchart.github.io](https://reprobrainchart.github.io). RBC is also accompanied with a publication at [Shafiei G, Esper NB, et al., (2025). Reproducible Brain Charts: An open data resource for mapping brain development and its associations with mental health. Neuron.](https://doi.org/10.1016/j.neuron.2025.08.026)
